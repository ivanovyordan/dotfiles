# Dotfiles

Personal dotfiles for macOS (Homebrew) and Arch Linux (paru).

Includes configs for fish, oh-my-fish, neovim, tmux, git, starship, ranger, direnv, and poetry. Managed with GNU Stow.

# Installation

```
curl -o- https://raw.githubusercontent.com/ivanovyordan/dotfiles/master/install/main.sh | bash
```
